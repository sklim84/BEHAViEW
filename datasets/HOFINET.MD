### 4.2 STR 양식 §VI 의심거래 분류 매핑 (FIU 공식)

HOFINET `이상거래유형`은 거래 데이터의 객관적 패턴 라벨이고, **STR 양식 §VI 의심거래 분류(코드 1-31)** 는 분석가가 STR 보고 시 선택하는 공식 분류다. 출처: `_docs/08_STR_보고서양식.md` §VI. STR 양식 §VI는 **복수 선택 가능**.

#### STR §VI 전체 sub-category 개요 (전체 31 코드)

| 서브 | 범주 | 코드 범위 |
|------|------|----------|
| VI-1 | 금융사고/긴급조사 | (체크) |
| VI-2 | 거래자 태도/특징 | 1-10 |
| VI-3 | 계좌정보 | 11-14 |
| VI-4 | 거래유형 | 15-30 |
| VI-5 | **기타(자유기술)** | **31** (catch-all) |

#### STR §VI-4 거래유형 (코드 15-30, 16종)

| 코드 | 거래유형 | HOFINET 1차 매핑 |
|------|---------|--------------|
| 15 | 갑작스러운 거래패턴의 변화 | **= HOFINET 1** |
| 16 | 원격지거래 | — |
| 17 | 교환거래 | — |
| 18 | 분할거래 | **= HOFINET 3** |

| 20 | 거액 입금 후 당일/익일 인출 | **= HOFINET 5** |
| 21 | 무기명증서 관련거래 | — |
| 22 | 계좌개설 없이 거액 환전/송금 | — |
| 23 | 의심스러운 담보대출/보험약관대출 | — |
| 24 | 주금 납입/잔액증명서 발급 | — |

#### HOFINET → STR §VI 1차/보조 매핑 (전체)

| HOFINET 코드 | 이상거래설명 | STR 1차 매핑 | STR 보조 매핑 |
|---|---|---|---|
| 1 | 갑작스러운 거래패턴의 변화 | **§VI-4 code 15** | — |
| 2 | 신규 수신처 거래 | **§VI-5 code 31 (자유기술)** | §VI-2 code 9 사전거래 없는 고객 |
| 3 | 분할 거래 | **§VI-4 code 18** | — |
| 4 | 다중거래의 동시 요청 | **§VI-4 code 25** | — |
| 5 | 거액 입금 후 당일 인출 | **§VI-4 code 20** | — |
| 7 | 심야/새벽 대량 거래 | **§VI-5 code 31 (자유기술)** | §VI-2 code 6 부조화조사 요망 |

**원칙**:
- HOFINET 1/3/4/5 → §VI-4 직접 매핑 (4종)
- HOFINET 2/7 → §VI-5 code 31 (자유기술) 매핑. 양식상 명시된 catch-all로 의도적 활용
- 보조 매핑은 분석가가 §VI 다중선택 시 추가로 표기 가능

`generate_str` 도구의 `fraud_type` enum은 **§VI-4 16종(코드 15-30) + §VI-5 code 31 = 17종**을 사용한다. 위 매핑 표에 따라 HOFINET 2/7 시나리오는 "기타(자유기술)"으로 자연스럽게 처리된다.
