# 의심거래보고(STR) 업권별 의심거래 참고유형 및 지표

FIU는 ML/TF 가능성이 높은 거래유형을 금융회사등에 제공하며, 최근 각 업권과 협업하여 의심거래 참고유형을 업데이트하여 제공(2022년 8월 발행).

| 7 | **고령자의 스마트폰** 등을 이용한 가상자산 거래 |
| 8 | **다수인들이 특정인에게** 가상자산을 출고하는 거래 |
| 9 | **법인과 대표자/직원**과의 가상자산 거래 |
| 10 | 고액의 가상자산을 반복적으로 **가족 추정인에게 출고** |
| 11 | 특정금융정보법상 각종 **보고 기준금액 이하로 가상자산 이전을 여러번** 하는 거래 |

## 10. 의심거래 지표 분류 요약 (벤치마크용)

의심거래 패턴을 탐지 로직 관점에서 분류하면 다음과 같다:

### 10.1 금액 기반 지표

| 패턴 | 탐지 조건 예시 | 관련 업권 |
|------|-------------|----------|
| 거액 현금 입출금 | 일정 금액 이상 현금 거래 빈발 | 은행, 증권, 보험 |
| 직업/소득 대비 과도한 거래 | 급여/매출 대비 거래금액 비율 초과 | 전 업권 |
| 고액 환전 | 일정 금액 이상 외화 환전 | 은행 |
| 고액 보험료 일시납 | 재산상태 대비 고액 보험 체결 | 보험 |

### 10.2 빈도/패턴 기반 지표

| 패턴 | 탐지 조건 예시 | 관련 업권 |
|------|-------------|----------|
| 분할거래 (Structuring) | CTR 기준금액 미만으로 반복 거래 | 전 업권 |
| 단기간 입출금 후 해지 | 계좌 개설 -> 거액 입출금 -> 단기 해지 | 은행, 보험 |
| 24시간/심야 대량거래 | 심야/새벽 시간대 포함 지속적 거래 | 은행 |
| 반복적 대출-상환 | 대출과 상환을 반복 | 보험, 대부 |
| 선결제-취소 반복 | 선결제 후 취소를 반복 | 여신금융 |

### 10.3 관계/네트워크 기반 지표

| 패턴 | 탐지 조건 예시 | 관련 업권 |
|------|-------------|----------|
| 타인명의 이용 | 미성년자/고령자/신용불량자 명의 거액거래 | 전 업권 |
| 제3자 상환/납입 | 경제적 합리성 없는 제3자가 대출 상환/보험료 납입 | 은행, 보험 |
| 법인-대표자 거래 | 법인 계좌와 대표자 개인계좌 간 거래 | 은행, 증권 |
| 동일 대리인 다수 계좌 | 동일인이 다수 계좌를 대리 개설 | 은행 |
| 불특정 다수와의 거래 | 관계 불명 상대방으로부터 입금 후 송금/현금화 | 은행 |
| 경유계좌 | A -> B(경유) -> C 패턴의 자금 이동 | 은행, 증권 |

### 10.4 가상자산 관련 지표

| 패턴 | 탐지 조건 예시 | 관련 업권 |
|------|-------------|----------|
| 거래소 경유 현금화 | 가상자산거래소 영수 -> 현금출금/해외송금 | 은행, 증권 |
| 다중 거래소 이용 | 복수 가상자산거래소를 이용한 증여/이전 | 은행, 가상자산 |
| 동일 IP 다수인 거래 | 동일 IP에서 다수인이 거래 | 가상자산 |
| 가격 차이 이용 | 국가 간 가상자산 가격 차(김치프리미엄) 이용 | 은행, 가상자산 |
| 저연령/고령자 거액거래 | 사회초년생, 고령자의 거액 가상자산 거래 | 가상자산 |

### 10.5 외환/무역 관련 지표

| 패턴 | 탐지 조건 예시 | 관련 업권 |
|------|-------------|----------|
| 분산 해외송금 | 5만 불 또는 1만 불 미만으로 분산 송금 | 은행 |
| 무역서류 불일치 | 수입신고 미이행, 수출대금 미결제 | 은행 |
| 원격지 외화 입출금 | 해외 ATM 이용 거액 외화현금 인출 | 은행, 여신금융 |
| 증여성 송금 | 실질적 송금 사유에 부합하지 않는 증여성 송금 | 은행 |

### 10.6 은닉/위장 관련 지표

| 패턴 | 탐지 조건 예시 | 관련 업권 |
|------|-------------|----------|
| 차명거래 | 타인명의로 거래를 위장 | 전 업권 |
| 위장법인 | 위장법인 또는 신설법인 계좌 이용 | 은행, 증권 |
| AML 제도 문의 | 자금세탁방지제도에 대해 필요 이상 문의 | 은행, 보험 |
| 비밀유지 요청 | 거래에 대한 비밀유지를 부탁하거나 정보제공 거부 | 은행, 증권 |
| 정보 불일치 | 고객정보와 실제 거래 패턴의 불일치 | 전 업권 |
