# 자금세탁 및 테러자금조달 위험의 이해 - 개요와 유형

## 1장. 자금세탁 및 테러자금조달 행위의 위험과 글로벌 유형

### 1.1 개요

- 우리나라는 2001년 9월 **특정 금융거래정보의 보고 및 이용 등에 관한 법률**(약칭: 특정금융정보법)을 마련하여 자금세탁방지제도를 법제화

---

### 1.2 자금세탁행위의 의미 및 자금세탁 3단계

#### (1) 배치단계 (Placement)

- 범죄활동에서 발생한 **범죄수익(현금 또는 기타 자산)을 이동**시키는 단계
- 자금세탁자는 불법수익을 금융시스템으로 유입시킴
- 최근에는 금융시스템 다변화로 **가상자산, 전자금융업자, 환전업자, 카지노** 등 다양한 금융회사를 통한 자금이동이 가능

**배치단계 예시:**
- 마약밀매 조직이 대량의 현금을 차량으로 밀반출 후 환전상을 통해 고객의 여행자수표와 고액권으로 교환
- 부피를 줄인 여행자수표와 지폐를 항공편으로 런던으로 반출 후 저지(Jersey) 섬에 본점이 있는 금융회사의 런던지점에 입금

#### (2) 반복단계 (Layering)

- 배치단계에서 금융회사등에 유입된 범죄수익에 대한 **자금의 출처를 숨기기 위해 다양한 방식의 금융거래를 반복**하여 불법적인 수익을 출처와 분리하는 단계
- 범죄수익의 출처나 소유자를 알 수 없도록 금융거래를 복잡하게 만드는 것
- 범죄자금과 합법 자금을 혼합하여 범죄자금 식별을 어렵게 함 -- 범죄자들이 선호하는 기업형태

**반복단계 예시:**
- 런던지점에 입금된 자금을 다시 저지본점에 개설된 수 개의 위장회사 명의의 계좌로 송금

#### (3) 통합단계 (Integration)

- 자금의 출처에서 분리된 범죄수익을 **정상적인 거래로 보이도록** 하여 정상적인 경제시스템에 자금을 재투입하여 **불법적인 범죄수익에 합법성을 부여**

**통합단계 예시:**
- 위장회사의 계좌에서 다시 인출된 자금은 마약조직이 소유한 미국 내 회사에 투자금 또는 대출금 명목으로 다시 반입

### 1.3 자금세탁행위에 자주 사용되는 개인 또는 법인 유형

#### 1. 무기명 주식을 보유한 법인

| 항목 | 내용 |
|------|------|
| 특징 | 일부 국가에서 기업이 무기명 주식 발행을 허용 |
| 위험 | 누가 보유하고 있는지 확인할 수 없어 실제 소유자 은닉에 효과적 |
| 한국 | 2014년 상법 개정으로 무기명 주식 발행 불가 |

#### 2. 위장회사 (Shell Company)

| 항목 | 내용 |
|------|------|
| 정의 | 서류 형태로만 존재하면서 회사 기능을 수행하는 회사 |
| 특징 | 물리적 실체가 없는 회사, 물리적인 주소/직원/실질적인 사업이 없음 |
| 사례 | 조세회피처인 케이만아일랜드에서 수백 개의 위장회사가 동일한 주소에 등록, 직원이나 실질 비즈니스는 없음 |

#### 3. 전위 사업체 (Front Company)

| 항목 | 내용 |
|------|------|
| 차이점 | 위장회사와 달리 **사업의 실질을 갖고 있음** |
| 위험 | 범죄자금과 합법 자금을 혼합하여 범죄자금 식별을 어렵게 함 |
| 사례 | 불법도박 사이트 조직이 불법도박 자금을 토대로 수개의 주유소를 구입하여 운영. 대부분의 수입은 불법도박 자금이며, 해당 자금은 주유소 자금에 혼합 |

### 1.4 자금세탁행위등에 대한 국제 공동의 위험지표 (Red Flag)

#### (1) 거래개시 또는 일회성 거래 시점에 의심스러운 고객의 거래활동

**비정상적인 행동을 보이는 고객:**

| Red Flag | 상세 |
|----------|------|
| 주변 살핌 | 고객이 거래시점에 주변을 살피거나 타인의 주도하에 거래를 수행 |
| STR/CTR 문의 | 보고의 기준 금액에 대해 상세히 문의하고 영향을 자세하게 문의 |
| 위협 | 거래를 CTR/STR 보고할 경우 금융회사 임직원의 신변에 위협 |
| CTR 회피 | CTR 기준 금액 초과 시 거래를 꺼려함 |
| 비합리적 환매 | 특별한 사유 없이 고금리 투자상품을 손해를 보고 환매 |
| 소득 불일치 | 주부/학생 등 소득이 없는 고객이 비정상적 거래 행태 |
| 역외 연관 | 조세회피처 등에 소재하는 자와 연관 |
| 다수 대리 | 한 사람이 여러 명의 고객을 대리하여 계좌 개설 |

**CDD 과정에서 비정상적 행동:**

| Red Flag | 상세 |
|----------|------|
| 불량 서류 | 사진 식별 불가 신원확인서류, 위조 가능성 있는 서류 |
| 주소 이상 | 역외에만 소재하거나 공유오피스이거나 실체 없는 주소 |
| 연락 불가 | 콜백 시 사용하지 않는 연락처이거나 착신불가 설정 |
| 정보 거부 | 매출금액, 예상 거래규모, 추가 자료, 직업/소득 정보 제공 거부 |
| IP 불일치 | 고객의 IP 주소가 허용되지 않은 영역이거나 식별정보와 불일치 |
| CSP 법인 | 기업설립서비스회사(CSP)를 통해 설립한 법인 |

#### (2) 거래수단이 비정상적인 유형

**현금 이용 비정상적 거래:**

| Red Flag | 상세 |
|----------|------|
| 미확인 현금 | 포대에 담긴 현금 또는 카운트하지 않은 상태로 입금 |
| 골프가방 현금 | 고액자산가가 골프가방에 거액의 현금을 갖고 와서 입금 |
| 소액권 교환 | 소액권을 고액권으로 교환하거나 거액의 수표를 분할 후 현금 인출 |
| 위조화폐 | 입금한 현금 중 위조화폐, 구권, 더러운 지폐 포함 |
| CTR 분할 | 여러 명의 명의를 이용하여 CTR 금액 미만으로 나눠 입금/인출 |
| 야간금고 | 비즈니스 활동에 부합하지 않게 야간금고를 이용하여 거액 예치 |
| ATM 분할 | 자동화기기를 통해 보고 기준금액 미만으로 여러 차례 현금 입금 |
| 원격지 거래 | 노령자의 계좌에서 원격지 ATM을 통해 수시로 거액의 현금 거래 |
| 외화 비정상 | 특별한 이유 없이 고액의 외화 현찰을 매입하거나 손해를 보며 매도 |

**수표/전신송금 이용 비정상적 거래:**

| Red Flag | 상세 |
|----------|------|
| 출처 불명 수표 | 출처가 불분명한 연속번호 여행자수표/자기앞수표 입금 |
| 규모 불일치 | 비즈니스 규모에 맞지 않는 가계수표/당좌수표 입금 |
| 다중 계좌 | 동일 인물이 서로 다른 계좌를 이용해 전신송금 |
| 고위험 지역 | 명백한 사업상 이유 없이 조세피난처/고위험 지역에 전신송금 |
| 소액 집적 | 여러 건의 소액 전신송금을 수취 후 다른 국가로 고액 송금 |
| 무기명 증권 | 무기명 증권 지급 후 즉시 제3자에게 송금 |

#### (3) 다양한 상품을 이용한 비정상적 거래 유형

**대출상품 이용:**
- 자금 여유가 있음에도 예금담보대출/무기명양도성예금 담보 대출
- 해외에 있는 예금을 대출 담보로 제공
- 시설자금 용도 대출금을 역외 법인에게 해외 투자 용도로 송금
- 중도상환수수료를 부담하면서 거액의 대출금을 단기간 내 상환

**기업 사업용 계좌 이용:**
- 재무제표가 유사 업종과 현저하게 차이
- 자본잠식 상태의 법인이 과도하게 거액 송금/무역거래 요청
- 거액 자금거래가 3~6개월 발생 후 거래 없는 계좌 (부가세 탈루 목적)
- 비즈니스 유형에 비해 지나치게 많은 계좌 보유
- 거래 상대방이 위장회사라는 의심 (동일 주소, 없는 주소 등)

**무역금융 거래 이용:**
- 수출/수입거래 시 인보이스 가격이 시장가격보다 현저히 높거나 낮음 (Over/Under Invoice)
- 수출거래 후 자금 결제가 해외 거래 상대방이 아닌 자기결제
- 수출 대전을 지속적으로 동일한 제3자에게 지급
- 소규모 자본금의 수입업체가 비정상적인 원유/금괴 수입
- 합리적 정당성 없이 신용장 수정, 직전에 수입상 변경
- 자금세탁 위험이 높은 국가로부터 발행된 신용장

#### (4) 해외 송금/환전 이용 비정상 거래

- CTR 보고 기준 금액 미만으로 거래 수행 (구조화)
- 서로 다른 거래의 증빙자료에 동일한 신분증이 자주 사용
- 브로커가 여러 명을 모집하여 거래
- 한 사람이 다수인을 위해 대신 거래

#### (5) 가상자산 이용 비정상 거래

| Red Flag | 상세 |
|----------|------|
| 트래블 룰 회피 | 100만 원 미만으로 거래하여 트래블 룰 회피 |
| 약한 규제국 | AML 법규가 취약하거나 트래블 룰 미적용 국가의 거래소만 거래 |
| 다수 이체 | 불특정 다수의 거래자가 한 명의 거래자에게 가상자산 이체 |
| 고령자 심야 | 고령자 거래자가 야간/심야시간에 빈번하게 이체거래 |
| 동일 IP | 동일한 IP를 이용한 다수의 거래자가 이체거래 요구 |
| 다크코인 | 다크코인이나 유의 종목 코인만 보유 |

#### (6) 보험상품 이용 비정상 거래

- 일시납 보험계약을 현금으로 입금, 정기납부 형태를 일시에 현금 납부
- 고액 정기납이 제3자에 의해 납부, 거래 관련인이 아닌 제3자가 현금으로 납부
- 장기 고액 보험료를 가입 후 수익자를 자녀 명의로 변경하고 부모가 계속 납부
- 거액의 일시납 보험료 납부 후 손실을 보며 해지

#### (7) 증권상품 이용 비정상 거래

- 수개의 증권계좌에 불특정 다수인으로부터 지속적인 거래
- 장기 미사용 계좌에 갑작스럽게 거액의 이체활동
- 장기 투자 목적으로 매수한 고객이 갑자기 매매
- 저가의 투기성 주식/유의종목 등에 거액 거래

#### (8) 귀금속/고가품 이용 비정상 활동

- 다이아몬드 생산량이 한정되거나 광산이 없는 국가가 원산지인 다이아몬드 거래
- 유명 귀금속 업체에서 매일 거액의 현금거래
- 규모가 작은 보석상에서 금괴, Loose Diamond 판매
- 제3자를 이용하여 다이아몬드 딜러 계좌로 자금 입금

#### (9) 카지노사업자의 비정상적 활동

- CDD/CTR 보고 기준 미만으로 게임
- 현금/자기앞수표로 칩 교환 후 게임에 일부만 사용하거나 손해 후 칩을 모두 현금 인출
- A국가 카지노에서 칩 구매 후 남은 칩을 해외 카지노에서 인출 요청
- 카지노 신용한도를 불법수익으로 상환
- 정킷(Junket)을 이용한 자금세탁: 에이전트를 통해 고객 모집, 자금의 출처/신원확인 취약성

### 1.5 FATF 자금세탁 및 테러자금조달 위험 유형

| 유형 | 내용 |
|------|------|
| 테러자금조달 크라우드펀딩 | 소셜미디어와 크라우드펀딩 플랫폼을 활용한 테러자금 조달 |
| 랜섬웨어 자금조달 | 범죄자들이 대부분 암호화폐/가상자산을 통해 자금 이동. AML/CFT 통제가 취약한 국가가 통로로 이용 |
| 미술품/골동품 시장 | 의심스러운 활동을 식별하는 위험 지표 목록 포함 |
| 펜타닐/합성 아편유사제 | 대부분의 국가에서 마약 밀매를 ML의 주요 전제범죄로 규정. 다크웹/가상자산에 대한 위험 인식 필요 |
| 인신매매 재정적 흐름 | 금융 정보를 통한 성적 착취/강제 노동 인신매매 식별. 장기 밀매 이익 창출 |
| 현금의 물리적 운송 | 국경을 넘어 현금을 물리적으로 운송하는 오래된 자금세탁 형태. 60개 이상 국가 정보 분석 |
| 불법 야생동물 밀수 | 초국가적 조직 범죄. 자금통로 추적이 범죄수익 환수에 가장 효과적 |

## 2장. 한국의 자금세탁 및 테러자금조달 위험

### 2.1 개요

- 각 국가는 자국의 ML/TF 위험을 확인, 평가 및 이해하여야 하며, **위험기반 접근법(RBA)**을 적용해야 함
- 정부는 2017~2018년 관계기관 합동으로 ML/TF 위험평가(국가위험평가) 실시, **9개 주요 자금세탁 위험** 확인
- 한국은 지정학적 위치와 문화적 특성 때문에 테러와 테러자금조달이 빈번하게 발생하지는 않음

#### 국가위험평가의 주요 개념

| 개념 | 정의 |
|------|------|
| **위협** | ML/TF와 관련된 전제범죄. 범죄행위 이외에 사람, 집단, 대상 등도 위협. 전제범죄와 범죄수익을 발생시키는 사회환경 이해가 중요 |
| **취약성** | ML/TF의 기회를 제공하는 제도, 기관, 분야, 규정, 법규, 감독, 법집행, 특정 서비스/상품 등. AML/CFT 제도의 약한 부분 |
| **결과** | ML/TF가 초래하는 영향 또는 위해. 금융체제/금융기관/경제/사회에 주는 효과 |
| **위험** | 위협, 취약성, 결과의 종합. 위협이 지속되거나 취약성이 해소되지 못할 때 ML/TF 위험으로 발현 |

### 2.2 한국의 주요 자금세탁 위험 (전제범죄 7종 + 취약분야 2종)

#### 전제범죄에서 확인된 7대 위험

| 위험 | 확인 배경 |
|------|-----------|
| **탈세/조세포탈** (관세포탈 포함) | 가장 빈번하고 광범위한 자금세탁행위. 지하경제 고려 시 매우 큰 규모. 국가재정 건전성에 심각한 위협 |
| **불법도박 등 불법사행행위** | 범죄조직과 연계, 규모 대형화. 참여자 경제적 손실, 가족/직장/건강 위협 |
| **보이스피싱 등 금융사기** | 불법 목적을 가진 조직된 범죄 집단에 의한 범죄. 여성/노동자 대상 지속 발생, 심각한 위협 |
| **부패범죄** (수뢰/증뢰/알선 등) | 국가권력을 이용한 악의적 범죄. 대규모 범죄수익 창출. 경제활동 왜곡 등 심각한 사회적/경제적 피해 |
| **주가조작 등 불공정거래** | 투자자 시장불신과 자본시장 건전성 위협. 건당 대규모 범죄수익. 더욱 증가하는 심각한 금융범죄 |
| **재산국외도피** (무역거래 악용 이용) | 허위 무역거래, 유령회사 등을 악용. 국가 재산 유출. 국가재정 건전성에 큰 위협 |
| **횡령/배임** | 기업의 존립과 다수의 경제활동에 영향. 한국의 취약한 기업구조가 요인. 범죄 파급력과 자금세탁 유인이 큼 |

#### 취약분야에서 확인된 2대 위험

| 위험 | 확인 배경 |
|------|-----------|
| **현금거래** | 탈세, 부패범죄, 자금세탁등의 주요 수단. 과거 고액권환수가 낮아 불법자금 은닉수단 의심. 일부 부동산거래에서 여전히 선호 |
| **가상통화(자산)** | 급격하게 성장, 규제가 취약. 거래 추적 곤란 악용, 범죄수익 모집 수단. 외국 불법자금의 국내 반입 통로 의심 |

### 2.3 주요 전제범죄 상세 분석

#### (1) 탈세/조세포탈

- 조세포탈이란 사기나 부정한 행위로 납부하여야 할 조세를 면하거나 환급/공제를 받는 범죄
- FIU가 STR을 분석하여 제공하는 범죄 유형별 건수에서 **탈세/조세포탈이 가장 큰 비중**

**주요 조세포탈 수단:**
1. 현금거래
2. 차명계좌
3. 부정세금계산서 발생
4. 장부 및 증빙서류의 허위작성 및 변조

**주요 조세포탈 유형과 사례:**

| 유형 | 사례 |
|------|------|
| 현금매출 누락/차명계좌 | 성형외과 등에서 현금결제 시 할인 유도 후 지인 명의 차명계좌로 입금, 고의적 신고 누락 |
| 부정세금계산서 수수 | 실지 용역 제공 없이 용역 대금 지급 후 현금/제3자 계좌로 되돌려 받는 방법으로 가공세금계산서 수취 |
| 금지금 업체 공모 | 순도 99.5% 이상 금괴 연속거래에서 폭탄업체로부터 도관을 거쳐 유통된 폭탄금을 매입하며 부가가치세 부당환급 |
| 이중계약서/미등기 전매 | 양도소득세 신고 시 양도가액을 낮춘 허위계약서(다운계약) 작성/제출 |
| 증빙서류 허위작성 | 출장비, 노임대장 허위작성 등 거짓 문서 작성 후 장부를 거짓 기장 |
| 명의위장 조세회피 | 인터넷 브로커로 대포사업자/대포통장 구입 후 오픈마켓에서 판매, 부가가치세 미납부 |
| 고리대금업자 차명계좌 | 이자대금 은폐 목적으로 현금/수표로만 자금 수수, 장부/금전대부약정서 폐기 |

#### (2) 불법도박 등 불법사행행위

| 구분 | 불법사행행위 유형 | 관련 법률 |
|------|------------------|-----------|
| 불법사행사업 | 불법 카지노 | 관광진흥법 |
| | 불법 경마 | 한국마사회법 |
| | 불법 경륜/경정 | 경륜/경정법 |
| | 불법 복권 | 복권 및 복권기금법 |
| | 불법 스포츠도박 | 국민체육진흥법 |
| 불법사행산업 | 불법 소싸움 | 전통 소싸움경기에 관한 법률 |
| | 사행성게임장 | 게임산업진흥에 관한 법률 |
| | 온라인도박 | 게임산업진흥에 관한 법률 |
| 불법사행산업 이외 | 불법 하우스도박 | 형법(246~247조) |
| | 경견/투견/투계 | 형법(246~247조) |
| 불법도박 | 기타 실내낚시 | 형법(246~247조) |
| | 일시 오락 범위 벗어난 도박 | 형법(246~247조) |

- 불법도박이 의심되는 계좌 발견 시 거래를 즉시 중단하는 조치가 필요하나, 현재 AML 관련 법규에서는 CDD 미수행 경우로 한정
- 최근 FATF에서 STR 보고계좌에 대한 **거래중단(Suspension)** 등의 적극적인 조치를 권고

#### (3) 보이스피싱, 보험사기 등 금융사기

**보이스피싱 유형:**

| 유형 | 수법 |
|------|------|
| 기관사칭형 | 검찰/경찰/금감원 사칭, 피싱사이트 유도, 현금인출기로 유인하여 직접 송금 유도 |
| 납치/공갈형 | 가족을 납치했다며 송금 유도 |
| 대면접촉/절취형 | 예금보호 명목으로 직접 만나 편취하거나 특정 장소에 보관 지시 후 절취 |
| 대출빙자형 | 대출을 빙자하여 개인정보 취득 후 대포통장으로 이체하거나 수수료 명목 금융 송금 유도 |

- 보이스피싱 범죄자들은 **타인명의 계좌(대포계좌)**를 이용하여 범죄수익을 이동
- 유사수신행위: 금융당국 인허가 없이 불특정다수 대상 원금/수익 보장하며 자금 모집 (폰지사기와 유사)
- 리딩방 투자사기: 유명인 사칭, 고령자에게 고수익 보장하여 거액 투자 유도
- 보험사기: 보험사고의 발생/원인/내용에 관해 보험자를 기망하여 보험금 청구

#### (4) 부패범죄

- 부패방지 전담기구(국민권익위원회) 설치
- 부정청탁 및 금품등 수수의 금지에 관한 법률 제정

#### (5) 주가조작 등 불공정거래

| 범죄 유형 | 내용 |
|-----------|------|
| 시세조종행위 | 위장거래, 허위표시, 현물/선물 연계 등 |
| 미공개정보이용 | 내부자거래행위 |
| 부정거래 행위 | - |
| 신고/공시의무 위반 | - |

#### (6) 재산국외도피

- 한국은 산업구조에서 무역 비중이 높고 수출입이 활발하여 **무역거래를 가장한 자금세탁에 취약**

**주요 범죄 사례:**
1. 수출대금 등을 미회수한 뒤 미신고 해외계좌에 예치 후 국내반입
2. 불법 해외투자를 통한 주가조작
3. 중계무역을 가장한 재산국외도피
4. 수출가격 조작을 통한 재산도피
5. 허위서류 작성을 통한 외화송금/재산도피

#### (7) 횡령/배임

- 다른 재산범죄와 비교할 때 **범죄수익규모가 상대적으로 크고** 자금세탁 유인이 높음
- 범죄로 인하여 영향을 받는 사람도 많고 사회의 도덕성과 사기 저하

**사례 유형:**
1. 금융회사로부터 불법 대출을 받아 횡령
2. 무자료거래, 회계조작 등을 통해 기업 자금을 횡령
3. 계열사 주식 헐값 인수를 통한 기업손해 발생 및 부당이득 수취
4. 부동산 개발 분양대금 횡령

**제도적 대응:**
- 외부감사법 전면 개정: 주기적 감사인지정제도 도입 등 회계제도 개선
- 상장적격성 실질심사제도: 코스닥 상장기업 중 횡령/배임, 분식회계 등 불건전행위 발생 기업에 대해 퇴출 여부 판단
  - 직원 횡령/배임 혐의 금액이 자기자본의 5%(대기업 3%) 이상
  - 임원 횡령/배임 혐의 금액이 자기자본의 3% 또는 10억원 이상

### 2.4 금융업과 거래수단에서 확인된 위험

- **현금거래**: 탈세, 부패범죄, 자금세탁등의 주요 수단
- **가상화폐(자산)**: 급격한 성장, 취약한 규제, 거래 추적 곤란, 범죄수익 모집 수단, 외국 불법자금의 국내 반입 통로 의심
