# 자금세탁 및 테러자금조달 위험의 이해 - 업권별 위험요소

## 3장. 각 금융회사등의 업권별 자금세탁 위험에 대한 이해

### 3.1 개요

- 한국의 자금세탁방지 의무를 준수하고 검사대상이 되는 "금융회사등"은 약 **9천여 개**
- 업권: 은행, 금융투자업, 보험업, 상호금융업, 여신전문업, 대부업, 소액해외송금업, 환전업, 온라인투자연계금융업, 벤처캐피탈업, 카지노사업, 전자금융업, 가상자산거래소

---

### 3.2 은행

#### 은행의 자금세탁위험 유형

- AML을 위한 내부통제 절차가 어느 정도 정착된 업권
- 상품 및 채널의 다양성과 취약점 등을 이용해 여전히 **은행을 자금세탁 통로로 가장 쉽게 이용**
- 가상자산거래소나 전자금융업자와의 거래로 인한 새로운 위험
- 다양한 국가로의 영업 확대로 해외점포의 AML 통제 강화 필요
- 총 20개 국내 은행: 시중은행 6개, 지방은행 6개, 특수은행 5개, 인터넷 전문은행 3개

#### 은행 상품별 자금세탁 취약점

##### (1) 예금상품

| 취약유형 | 상세 |
|----------|------|
| **차명계좌** | 법인의 비자금 이동, 탈세 채널로 이용. 계좌 개설 시 실소유자 여부 확인, 자금의 출처 파악 필요 |
| **대포계좌** | 보이스피싱에 활용. 금융거래확인서로 거래목적 확인/검증, 미확인 시 한도계좌로 통제. 범죄자들은 적립식 예금, 자동이체 등록 채널을 이용한 거액 이체 등 새로운 방식 사용 |
| **무기명 양도성 예금증서** | AML에서 우선적으로 금지되어야 하는 익명성 상품. 한국에서는 무기명식 양도성예금증서 발급 허용. 최초 발급의뢰인과 최종 소지인은 확인 가능하나 중간 양도 확인 불가. 불법사채 시장에서 할인 판매, 중간 보유자 은닉 |
| **가상자산 연계 원화 실명계좌** | 거래소는 지정된 은행으로부터 원화실명계정을 제공받아야 함. 은행은 불법적인 자금의 가상자산거래 여부 파악을 위한 EDD/모니터링 수행 필요 |

##### (2) 대출상품

- 자녀명의 대출을 부모가 상환하는 방식으로 증여세 탈루
- 뇌물 제공자가 대출자금을 변제
- 대출금 조기 변제 시 또는 소득 대비 고액 부동산 매입 후 제3자/현금으로 상환되는 경우 모니터링 필요
- 공공데이터 스크래핑을 통한 소득증빙 이용으로 위조 대출사기 감소

##### (3) 외환상품 (무역금융거래 포함)

| 취약점 | 상세 |
|--------|------|
| 역외 자금이동 용이 | ML 리스크가 높은 상품 |
| 증빙자료 검증 어려움 | 외국환거래법에서 증빙자료 요구하나 진위 여부 검증 용이하지 않음 |
| TBML 시스템 | 2022년 허위 무역서류로 16조 이상의 해외송금 발생 후 TBML 시스템 도입 추진 |
| 여행사 환전 | 허위 증빙서류 작성, 동일 신분증이 반복 발견 |
| 비대면 소액 해외송금 | 거래 목적/자금 원천 증명 불필요한 간소화된 역외자금 이동. 하루에 수백 번 건별한도 미만 거래, 거액 자금 이동 채널로 이용 |

##### (4) 환거래(계약) 서비스

- 환거래은행이 환거래요청은행에게 자금관리/수표결제/국제간 자금이체/무역관련 서비스 등 제공
- ML/TF 위험이 높아 AML 통제 강화 대상

##### (5) 자동화기기(ATM) 채널

| 취약점 | 상세 |
|--------|------|
| 익명성 | 거래 수행자 확인 불가 |
| 자금 출처 불명 | 거액 현금 인출/입금 시 자금 출처/용도 파악 불가 |
| 보이스피싱 | 범죄자들이 ATM을 이용해 현금 인출하는 경우가 많음 |
| 입금 통제 부재 | 입금의 경우에는 별도 통제가 없어 개선 필요 |

##### (6) 대여금고

- 고액자산가들의 재산은닉 수단으로 사용 가능
- 보관대상에 제한 없음 (귀금속/보석류, 권리증서, 채권/기타증권, 양도성 예금증서, 예금통장 등)
- 고객 위험도에 따라 대여금고 서비스 제공 여부에 대한 내규 마련 필요

##### (7) 가상계좌

| 항목 | 내용 |
|------|------|
| 정의 | 은행이 제공하는 펌뱅킹 서비스. 다수 고객과 거래하는 사업자가 고객에게 가상계좌를 부여, 수납대금이 모계좌에 자동 이체 |
| 대상업체 | 보험사, 카드사, 할부금융사, 쇼핑몰, 백화점, 학교 등 |
| **악용사례** | 범죄 일당이 쇼핑몰 영업으로 서류를 꾸며 가상계좌 발행 권한을 받은 후 **6만 4천여 개를 찍어내** 도박사이트나 전화금융사기 집단에 판매 |
| 통제 필요 | 사업목적 비추어 필요 여부, 과거 보이스피싱/도박 관련 이력, 업체 규모 대비 과도한 발급 여부 점검. PG사의 재판매/재재판매는 강화된 실사대상, 재재판매는 원칙적 불허 |

### 3.3 금융투자업

#### ML/TF 취약점

##### (1) 증권 매매거래를 통한 자금세탁

- **주식실물 출고**: 평소 거래가 빈번하지 않던 고객이 거액의 유가증권 매입을 현금 결제 후 실물인도 요청
- **주식매도대금 현금 인출**: 안정적 성향 고객이 갑자기 기존 계좌 해지, 신규 계좌로 주식거래 시작 후 전량 매도
- **타인명의 계좌 사용**: 계좌명의인이 아닌 타인에 의해 계좌가 운용되는 것을 의심

##### (2) 국경간 주식거래를 통한 자금세탁

- **위장 외국인 불공정거래**: 해외 위장회사를 통해 외국인투자자로 위장하여 시세조종, 미공개 내부정보 이용
- **탈세**: 주식 양도차익 과세 회피, 법인세/종합소득세 회피
- **비자금 조성**: 수출입 거래 조작, 증권 불공정거래를 통해 해외 페이퍼컴퍼니에 자금 은닉

##### (3) 증권시장 불공정거래

- **계좌간 통정매매**: 다수 명의로 여러 지점에 계좌 개설 후 가장/통정매매로 시세차익 추구
- **주가관리**: 일일 거래량이 적은 특정 주식을 수개월 걸쳐 지속 매매 후 전량 매도, 대금 현금 인출
- **미공개정보 이용**: 계좌 개설 당일 주식 입고하여 급하게 처분, 익일 악재성 공시로 주가 급락
- **장기 미거래 후 거액 입금**: 거액 자기앞수표 입금 후 제3자를 주문대리인으로 지정, 코스닥 종목 집중 매수 후 급등 시점에 전량 매도

### 3.4 보험업

#### ML/TF 취약점

- 현금거래가 빈번하지 않고 장기성 상품을 취급하여 다른 금융권에 비해 ML 위험이 낮은 편
- **그러나**: 최근 고금리 기조에 따른 장기 고금리 고액 보험상품 출시, 자금 출처 불분명한 거액 현금 가입 사례 발견

**보험상품 관련 자금세탁 사례:**

| 유형 | 수법 |
|------|------|
| 보험계약 악용 | 고액 일시납 후 단기 해약. 법인 대표자가 보험을 통해 비자금 조성/횡령/탈세. 부모가 자녀명의 중도인출 후 계약자 변경 |
| 보험약관대출 이용 | 고액 보험계약을 실효/부활시킨 후 고액 보험계약대출 실행, 다시 보험계약 실효 |
| 보험사기 | 자살, 살인 등 고의사고, 허위 과다사고 |

### 3.5 저축은행/상호금융업

#### ML/TF 취약점

| 취약점 | 상세 |
|--------|------|
| 직원 횡령 | 지속적으로 증가. 폐쇄성이 강하고 직무분리/순환근무 등 내부통제가 느슨, 범죄자금 환수율이 낮음 |
| 대주주 유용 | 대주주에 의한 고객자금의 유용 가능성 높음. 실질 대주주의 일가 소유 법인과 지인 중심 ML 의심 거래 |
| 시스템 한계 | 중앙회 시스템 사용이 일반적. CDD 위험평가 모형/STR 룰 개선이 적시 이루어지지 않을 가능성 |

- 새마을금고: 2023년 12월 기준 1,288개. 오프라인 지점이 영업 중심, 디지털화 뒤쳐짐
- 신용협동조합: 2022년 말 기준 870개

### 3.6 여신전문업

- 2023년 10월 기준 158개 (신용카드사 9개, 리스금융사 50개, 신기술 금융사 99개)
- 수신업무 미취급으로 ML/TF 위험이 현저히 낮으나 신용카드/대출업무가 ML에 이용 가능

| 취약유형 | 상세 |
|----------|------|
| **카드깡** | 기업/개인사업자가 유령 카드가맹점 매출을 꾸며 세금 회피. 온라인쇼핑몰에 유령판매점 만들어 이용 후 폐업. 소액 급전 서민에게 대출사기 유발 |
| **선불카드** | 기업이 마케팅 목적으로 대량 구입 후 매도하여 현금 마련, 뇌물공여/리베이트 등 불법 목적 |
| **가상자산 거래** | 2018년 1월부터 신용카드/체크카드를 이용한 가상자산 거래 차단. 지속적 모니터링으로 가상자산거래소 가맹점번호 확인 시 차단 |
| **트래블 카드** | 해외에서 현금 인출에 이용 |
| **직불카드 해외 이용** | 해외에서 현금을 지속적으로 인출 |

### 3.7 대부업

- 2024년 12월 기준 만 여 개. 기본적으로 고이율 소액 대출 경향, ML 위험이 낮은 것으로 판단
- FATF 요구에 따라 2019년 특정금융정보법 시행령 개정, **자산규모 500억 원 이상**의 대부업자에게 AML 의무 부과

### 3.8 소액해외송금업

- 2024년 12월 기준 31개 (독립형)
- 건당 5,000달러, 동일인당 연간 지급/수령 누계한도 각 5만 달러
- 등록 시 사전에 정한 금융회사의 계좌를 통해 거래해야 함

#### ML/TF 취약점

| 취약점 | 상세 |
|--------|------|
| 풀링(Pooling) | 여러 건의 송금신청을 모아 한 번에 송금 |
| 프리펀딩(Pre-funding) | 외국 협력업자에게 미리 목돈 송금 후 해외송금 접수 시 현지에서 동 자금 지급 |
| 비거주자 거래 | 외국인 근로자 등 비거주자 거래가 많아 테러자금 조달 이용 가능성 높음 |
| 핀테크 발달 | 가상자산을 이용한 다양한 송금 가능성, 모니터링 강화 필요 |

### 3.9 환전업

- 2024년 11월 기준 1,437개 업체 등록
- 환전업 미등록 수행 시 형사처벌 대상

#### 환전업자 종류

| 유형 | 특징 |
|------|------|
| 일반(카지노) 환전 | 등록된 영업장 내에서만 환전. 외화 매입 금액제한 없음. 외화 매각 미화 2천불 이하 |
| 온라인 환전 | 인터넷으로 환전신청 후 영업소 이외 장소에서 거래. 매입/매각 모두 미화 2천불 이하 |
| 무인환전기기 | 신분증 스캔 기능. 매입/매각 모두 미화 2천불 이하 |

#### 자금세탁 취약성

- 은행보다 높은 우대환율 제공, 외국인이 많은 거점에 위치하여 접근 용이
- 영세성, **환치기** 거래 증가, 가상자산을 이용한 불법 환전거래 증가, 미등록 환전업자 불법거래 등에 악용

**환치기 수법:**
1. 무등록 외국환 업무로서
2. 해외로 실제 송금 행위 없이
3. 국내 환치기 업자의 계좌에 입금하면
4. 국외 환치기업자가 입금 확인 후
5. 해당금액을 해외에서 송금목적인에게 지급

#### 주요 위반 사례

| 사례 | 내용 |
|------|------|
| 거액 자금 쪼개기 | 일본인 운반책이 밀반입한 거액의 엔화를 CDD 없이 환전, 보관중이던 다수 여권사본으로 소액으로 쪼개고 환전장부 허위기재 |
| 가상자산 이용 신종 환치기 | 중국에서 위안화 수령 후 중국 가상자산 구매, 국내 거래소로 전송 및 매도하여 원화 현금화 후 지정 국내계좌에 이체 |
| 비대면 채널 악용 | 외국인/비거주자가 비대면채널로 증빙서류 없이 해외 송금. 은행은 월간 1만달러 이하로 한도 축소 |
| 코인 환치기 | 해외 거래소에서 싼값에 비트코인을 사기 위해 송금 후, 국내거래소에서 팔아 차액을 남기고 해외로 빼내기 |

## 4장. 새롭게 대두되는 금융회사등의 자금세탁 유형

### 4.1 새롭게 대두되는 전제범죄

#### (1) 마약범죄

- 마약 판매 대금을 가상자산 또는 전자금융업자를 통한 소액이체서비스 등으로 이전
- **탐지 패턴**: 20만 원, 60만 원 등 끝전이 없는 다수 거래자로부터 소액 자금이 대규모로 입금

#### (2) 디지털 성범죄

- 소액 자금으로 이동, 대부분 가상자산(특히 다크코인), 소액이체 서비스 등을 통해 이루어져 모니터링이 어려움
- 정부는 디지털 성범죄를 전제범죄에 추가하여 중범죄로 처벌

#### (3) 가상자산을 이용한 시세조작/금융사기

- 유사수신행위, 폰지사기가 가상자산투자 명목으로 변화
- 가상자산 해킹 사건, 시세조작 공모
- 가상자산사업자는 Hot Wallet에 30% 미만, 나머지는 Cold Wallet에 보유하도록 가이드
- 2024년 7월부터 가상자산이용자보호법 시행

### 4.3 가상자산거래소

- 2025년 2월 기준 총 28개 가상자산사업자 갱신 신고 진행
- FATF는 2018년 10월 권고안 15(신기술) 개정, 가상자산 및 가상자산사업자에 대한 AML 규제 의무화

#### 가상자산거래소의 자금세탁 취약성

| 취약성 | 상세 |
|--------|------|
| **익명성** | 가상자산은 익명 거래 가능. 2021년 9월부터 CDD 도입. 2022년 3월 트래블룰 시행. 국내에서 규제가 약한 해외거래소로 이동 시 자금 추적 어려움 |
| **역외거래 가능** | 전통적 금융회사의 외국환거래법 통제와 달리 가상자산의 역외거래는 제한 없이 이동 가능 |
| **식별자 정보 부족** | 탈중앙화 성격으로 효과적인 관리감독 어려움. 지갑소유자의 체인거래 정보를 항상 검증하기 무리 |
| **보안취약성** | 해킹/전산장애 대비. Hot Wallet 보유한도 제한. 보험/공제 가입 또는 준비금 적립 요구 |
| **규제수준 상이** | 국가마다 통제환경 상이. FATF에서 국제기준을 정하고 이행 권고 |

#### 트래블 룰 (Travel Rule)

- 다른 가상자산사업자에게 **1백만 원 상당 이상**의 가상자산을 이전하는 경우 송금인/수취인의 실명, 가상자산주소 등 고객정보를 의무적으로 제공
- 수집된 정보를 거래관계 종료 시부터 5년간 보존
- 해외 가상자산사업자로 이전 시 해외사업자가 트래블 룰 의무를 부담하지 않는 경우가 많음
- 개인지갑으로 이전 시 사전등록제(화이트 리스트 제도)를 자율적으로 추진

### 4.4 무역거래를 이용한 자금세탁 위험 (TBML)

#### TBML 정의

- FATF TBML 보고서(2006): **불법자금의 원천을 합법화시키기 위한 목적으로 무역거래를 이용하여 범죄수익을 가장하거나 이동시키는 일련의 과정**
- 2017년 볼프스베르크 그룹과 BATF가 무역금융 원칙 가이드라인 발표

#### TBML을 위한 주의의무 강화 (Trade Due Diligence)

**고객확인 및 무역거래 주의의무:**
1. 해당 고객이 사업 성격상 무역거래 가능성이 존재하는지
2. 무역금융에 연관된 물품이 해당 고객의 사업에서 발생할 수 있는 물품인지
3. 경제제재조치 국가나 지역과 연관성이 없어야 함

#### TBML Red Flag (위험 지표)

##### 고객 관련 위험 지표

| Red Flag | 상세 |
|----------|------|
| 위장회사 관련 | 법인 구조가 위장회사와 관련되거나 고위험 국가에 등록된 회사처럼 복잡하고 비논리적 |
| 취약국 소재 | AML/CFT 규제가 취약한 국가에서 설립/등록 |
| 의심스러운 주소 | 대량 등록 가능한 건물, 사서함, 주거용 부동산을 사업장으로 사용 |
| 온라인 정보 부족 | 온라인 확인 정보가 부족하거나 사업 분야와 불일치 |
| 비즈니스 활동 부족 | 정기적 급여, 운영비용, 세금 납부 등 관련 거래 부족 |
| 명의인 의심 | 소유자/경영진이 경영 경험 부족, 여러 회사 관리 |
| 부정적 보도 | 조사 진행 중이거나 유죄 판결 이력 |
| 직원 규모 불일치 | 거래 상품 규모에 맞지 않게 최소한의 근무 직원 |
| 유명 기업 모방 | 상호가 유명 기업 이름을 복사한 것처럼 보이거나 유사 |
| 휴면기간 | 설명할 수 없는 거래 미발생 기간 |

##### 무역거래활동 관련 위험 지표

| Red Flag | 상세 |
|----------|------|
| 업종 불일치 | 무역 활동이 법인의 사업 분야와 불일치 (예: 자동차 딜러가 의류 수출) |
| 복잡한 거래 | 사업과 관련 없는 다수의 제3자 중개인을 포함하는 복잡한 거래 |
| 비정상 경로 | 표준 비즈니스 관행과 일치하지 않는 거래/운송 경로/방법 |
| 비전통적 금융상품 | 비정상적으로 긴 신용장 유효기간, 다양한 무역 금융 상품 혼합 사용 |
| 불합리한 이윤 | 소매가격과 비슷하거나 더 높은 가격으로 도매 수입, 구매가격과 같거나 낮은 가격으로 재판매 |
| 경제적 능력 초과 | 구매 금액이 기업의 경제적 능력을 초과 |
| 신생법인 대규모 거래 | 새로 설립된 법인이 대규모/고가치 무역 활동에 참여 |

##### 무역서류 및 상품 관련 위험 지표

| Red Flag | 상세 |
|----------|------|
| 서류 불일치 | 수출법인명과 자금 수령인 불일치, 송장/계약서 가격 불일치, 상품 수량/품질/가격 불일치 |
| 가격 이상 | 수수료/가격이 시장 가치와 불일치하거나 이전 거래와 크게 변동 |
| 모호한 설명 | 거래 상품에 대한 모호한 설명, 계약 주체가 구체적으로 기술되지 않음 |
| 서류 위조 | 누락/위조된 문서, 허위 정보 포함, 자주 수정/변경 |
| 과도하게 단순한 계약 | 인터넷 계약서 샘플 구조를 따르는 단순한 계약서 |
| 수출입 가격 불일치 | 등록된 수출입 가격과 실제 송금/수금 금액 사이의 상당한 불일치 |
| 위조 문서 수출 | 일시적 수입 국내 가공 후 위조 문서와 함께 수출 |
| 비합리적 운송 경로 | 경제적/상업적 정당성 없이 여러 국가를 경유하며 운송 |

##### 계좌 및 거래내용 관련 위험 지표

| Red Flag | 상세 |
|----------|------|
| 마지막 순간 변경 | 이전에 알려지지 않은 법인에게 자금 지급 변경 지시, 지급일/금액 변경 요청 |
| 과도한 거래 | 비즈니스 활동과 불일치하게 과도한 거래건수/금액 |
| 통과계좌 | 대량 거래의 빠른 이동, 소액의 당일 잔액, 이후 역외 법인/개인에게 현금 이체 |
| 제3자 지급 | 수입 상품 대금이 상품 수취인이 아닌 위장회사/전위사업체에 의해 지급 |
| CTR 미만 거래 | 계속하여 보고 기준 금액 미만으로 거래 |
| 급격한 변동 | 거래량이 빠르고 크게 증가 후 단시간 내 휴면 상태 (부가세 탈루 목적으로 3개월 단위 업체 변경) |
| 끝전 없는 금액 | 거액의 Round 금액으로 송금/수취. 무역거래는 일반적으로 끝전이 있는데, 무역사기/자금세탁의 경우 끝전 없는 거래가 많이 발생 |
| 순환 결제 | 한 국가에서 송금하고 다른 국가를 통과 후 다시 원래 국가에서 수취 |
