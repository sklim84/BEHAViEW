# 자금세탁방지 글로벌 기준 이해 (Part B) -- EU 지침과 주요국 감독기관

## 4장. 유럽연합(EU)의 자금세탁방지 지침(AMLD)

### 4.1 EU AMLD 발전 연혁 요약

| 지침 | 연도 | 핵심 변화 |
|------|------|-----------|
| **AMLD 1** | 1991 | 금융시스템의 자금세탁방지를 위한 법률 제정 요구, KYC 의무 부과, STR 최초 도입. 비금융부문 미포함. 마약밀매에 한정 |
| **AMLD 2** | 2001 | 전제범죄를 **모든 심각한 범죄**로 확대(부패/사기 포함). AML 의무를 **DNFBP**로 확대 |
| **AMLD 3** | 2005 | ML/TF 정의 및 범위 확대. **RBA** 도입. 법인 실제소유자 확인(25% 이상 지분). PEPs 강화 모니터링. 익명계좌 금지 |
| **AMLD 4** | 2016 | CTR 기준 **10,000유로**. 도박 서비스 확대. 조세범죄 포함. 국내외 PEPs 동일 정밀조사. 실제소유자 중앙등록부 관리. EDD 면제조항 삭제 |
| **AMLD 5** | 2018 | **가상자산(암호화폐)** 규제. 선불카드 요구사항 강화. 중앙등록시스템 연결/접근성 확대. FIU 역할 강화 |
| **AMLD 6** | 2024 | 디지털 금융환경 대응. 부동산/은행계좌 등록 접근 용이. 위험평가 강화. 실제소유자 투명성 강화. AMLA 주도 감독대학 설립 |

> **EU 지침은 회원국의 국내법보다 우선**

### 4.2 제1차 자금세탁방지지침 (EU AMLD 1, 1991)

- "자금세탁을 목적으로 금융시스템의 사용 방지" 채택
- 회원국들의 금융시스템에 AML 법률 제정 요구
- KYC 의무 부과, 의심스러운 거래 발견 시 FIU에 보고 (최초)
- 비엔나 협약 기준으로 마약밀매에 대한 자금세탁만 전제범죄로 한정
- 비금융부문은 적용 대상에 미포함

### 4.3 제2차 자금세탁방지지침 (EU AMLD 2, 2001)

- 전제범죄 범위를 마약밀매에서 **부패 및 사기를 포함한 모든 심각한 범죄**로 확대
- 자금세탁방지의 적용범위를 명시적으로 도입
- 자금세탁의 형사행위에 대해 객관적 사실에서 추론 가능함을 명확화
- AML 의무를 **특정비금융사업자 및 전문직(DNFBPs)** 영역으로 확대

### 4.4 제3차 자금세탁방지지침 (EU AMLD 3, 2005)

- FATF 권고안에 따라 ML/TF 정의와 범위 확대
- 범죄 파생 자금이 범죄조직 + 테러자금 지원에도 적용
- CDD 및 STR을 비금융사업자와 전문직에 확대 (15,000유로 이상 귀금속/고가품 거래업자, 회사설립 서비스 법인 포함)
- **위험기반접근방식(RBA)** 세부 반영

| 항목 | 내용 |
|------|------|
| 실제소유자 | 법인/개인이 **25% 이상** 지분 보유 시 확인의무 |
| CDD | 간소화된 CDD는 저위험 고객에 한정. 위험 인지 시 CDD/EDD 수행 |
| PEPs | 경영진 승인, 자금 출처 확인, 비즈니스 모니터링 강화 |
| 익명계좌 | 유지/통장보관 금지, 기존 익명계좌 소유자/수익자에 대해 CDD 실시 |
| 기록보관 | 거래종료 후 **5년**, 필요한 증거/참고자료 모두 보관 |

### 4.5 제4차 자금세탁방지지침 (EU AMLD 4, 2016)

| 변경 사항 | 상세 내용 |
|-----------|-----------|
| 고액현금거래보고 | 기준을 **10,000유로**로 강화 |
| 도박 규제 | 자금세탁 범위를 카지노에서 **도박 서비스 제공업체**로 확대 |
| 조세범죄 | 직접 및 간접세와 관련된 조세범죄를 범죄 행위 정의에 포함 |
| CDD 변경 | 10,000유로 이상 현금거래, 카지노에서 2,000유로 이상 칩 교환/당첨금 수령 시 CDD |
| PEPs | EU에만 한정되지 않고 **국내외 PEPs 모두 동일한 정밀조사** 명시 |
| 실제소유자 | **중앙 등록부서 관리** |
| EDD 면제 삭제 | AMLD 3의 공공기관/금융기관 등에 대한 간소화 CDD 면제조항 삭제. 위험평가 수행 후 단순화된 현장실사의 일부로만 적용 가능 |
| RBA 강화 | EU 지역 내 본사 기업의 역외 자회사도 본사에 준하여 지침 준수 |

### 4.6 제5차 자금세탁방지지침 (EU AMLD 5, 2018)

| 변경 사항 | 상세 내용 |
|-----------|-----------|
| **가상자산 규제** | 가상자산이 EU AML/CFT 적용범위에 포함. 모든 플랫폼/사업자에 KYC, AML/CFT 의무 |
| **선불카드 강화** | 익명성 최소화, 발행금액 월 150유로 초과 불가, 테러위험 높은 지역은 50유로로 제한 |
| **중앙등록시스템** | 모든 기업/법적 구조의 실제소유자 정보 등록. 일반 대중 접근 허용. 회원국 간 중앙등록시스템 연결 |
| **AML/CFT 기업 의무** | 자본/의결권 25% 초과 또는 영향력 있는 권한 행사하는 개인 등록 |
| **고위험 지역 거래** | 파트너에 대한 광범위한 정보 수집, 강화된 CDD, 거래목적 증명 요구 |
| **FIU 역할 확대** | 실제소유자 정보/거래내역 분석, STR 접수/분석/처리, 회원국 간 신속 정보 교환 |

### 4.7 형사법을 통한 자금세탁방지 지침 (Directive (EU) 2018/1673)

- 2020년까지 자국법 반영, 2021년 모든 회원국 준수 의무
- AMLD 4를 보완하여 회원국 간 사법 협력(형사법) 강화

| 항목 | 내용 |
|------|------|
| 전제범죄 확대 | 사이버 범죄, 환경 범죄, 탈세, 부정거래, 암호화폐 사기, 마약밀매, 인신매매 등 **22가지** |
| 법인 책임 강화 | 유한책임회사, 파트너쉽, 상장회사 등 법인도 직원 행동에 대해 AML 의무 위반 시 형사 책임 |
| 사법 협력 | 이중처벌 방지를 위한 관할권 명확화 |
| 범죄수익 몰수 | 법적 절차 간소화 |

### 4.8 제6차 자금세탁방지지침 (EU AMLD 6, 2024)

| 항목 | 내용 |
|------|------|
| **국가 메커니즘** | 불법 활동에 대한 금융시스템 오용 방지를 위한 강력한 시스템 구현 요구 |
| **정보 접근** | 부동산 정보, 실제소유자, 은행계좌 등록에 대한 감독당국 접근성 향상 |
| **위험평가** | EU 및 회원국 수준 모두에서 ML/TF 위협의 식별/평가 강조, 통합 접근 방식 |
| **실제소유자 투명성** | 실제소유자 등록/식별/확인 요구사항 강화. 언론인/시민사회단체 등 공공인사의 정보 접근 허용 |
| **부동산 등록부** | 신규 구축 -- 소유권, 거래가격, 이전 기록, 담보권 등 정보 포함 |
| **암호화폐 규제** | 암호화폐 자산 서비스 제공업체도 AML 규제 대상 포함 |
| **감독당국** | 국경을 초월한 의무기관에 대한 공동 감독을 위한 **감독대학** 설립 근거. AMLA가 주도적 지원 |
| **FIU** | 분석 역량 강화, 행정/금융/법집행 정보에 즉각적/직접적 접근, 불법자금 흐름 차단 권한, FIU 간 협력 강화, FIU.net 기능 향상 |

### 4.9 자금세탁방지청 (AMLA, 2024)

**설립**: 2024년 6월 26일 (Regulation (EU) 2024/1620)

EU AML/CFT 체계 강화를 위한 **초국가적 중앙감독기구**. 고위험 금융기관과 대규모 자산운용사에 대해 직접 감독 권한 보유.

**주요 기능**:

| 기능 | 내용 |
|------|------|
| 직간접 감독 | 암호화 자산 서비스 제공업체를 포함한 금융부문 고위험 의무기관에 대한 직간접적 감독 |
| 직접 감독 | 위험도가 높거나 국경을 넘어 운영되는 암호화폐 자산서비스 제공업체 포함 특정 유형의 신용/금융기관 직접 감독 |
| 국가 감독기관 협력 | 의무기관의 AML 준수 보장, 금융부문 AML/CFT 의무 대응, 비금융부문 지원 |

### 4.10 자금세탁방지규정 (AMLR, Regulation (EU) 2024/1624)

- 기존 수차례 지침(Directive)과 함께 **직접 적용 가능한 규정(Regulation)** 도입
- EU 회원국 전체에 **직접 적용** -- 각국 개별 법률로 전환할 필요 없음
- **2027년 7월 10일**부터 EU 회원국에 직접 적용 예정

**발표 목적**:
- 금융시스템 보호
- 통일된 법적 틀 제공
- 디지털 자산 포함
- 고객확인의무 강화 및 거래 보고 표준화
- 적용범위 확대 (부동산 중개인, 가상자산 서비스 제공자 등 비금융 업종)
- 데이터 공유 및 투명성 증대

## 5장. 주요 국가의 AML/CFT 감독기관 개요

### 5.1 미국의 금융감독기관

미국은 한국과 달리 **금융분야별 감독기구가 분리**된 체계를 운영한다.

#### 분야별 감독기관

| 분야 | 감독기관 | 비고 |
|------|---------|------|
| **은행** | FRB (연방준비제도이사회) | 통화정책, 금융위기 관리, 결제제도 안정성 |
| 은행 (FRS 가입) | FRB + 주정부 금융당국 | |
| 은행 (FDIC 가입) | FDIC (연방예금보험공사) + 주정부 금융당국 | |
| **은행지주회사** | FRB | 업무 전반에 대한 감독. FSOC도 위험 파악 수행 |
| **증권** | SEC (증권거래위원회) | 연방증권법 해석/집행, 증권회사/투자자문사/신용평가사 검사/감독 |
| **저축금융기관** | FRB, OCC, FDIC | |
| **보험회사** | 주정부 관할 | FIO(연방보험청)에서 모니터링. FDIC 가입 시 FDIC 감독도 추가 |
| **신용조합** | NCUA (전국신용조합관리국) | 국법 신용조합 인가/검사 |

#### 미국 재무부(DoT) 소속 AML 핵심 기관

| 기관 | 약어 | 역할 |
|------|------|------|
| **해외자산통제실** | **OFAC** | 테러리스트, 마약밀매자, WMD 확산 등에 대한 경제/무역 제재 관리/집행. **SDNs List** (특별지정 제재대상 국가 및 인물 목록) 발표 |
| **금융범죄단속네트워크** | **FinCEN** | 은행비밀법(BSA) 근거 금융범죄 예방. 금융정보 수집/분석. 금융기관 직접조사, 특정계좌 폐쇄 가능. **SAR(의심활동보고)** 30일 이내 제출 의무화 |
| **통화감독청** | **OCC** | 국법은행/연방저축기관 감독. 외국은행 미국 지점 허가/관리. 위험기반 감독절차 수행 |

> **FinCEN과 SAR**: BSA 및 AML법에 따라 금융기관은 30일 이내에 FinCEN에 의심활동보고(SAR)를 제출해야 한다.

### 5.2 유럽연합(EU) 감독기관

EU는 역내 일관된 금융감독 및 금융안정기능 강화를 위해 **유럽금융감독체계(ESFS)**를 구축했다.

#### 유럽감독당국(ESAs)

| 기관 | 약어 | 역할 | AML/CFT 관련 |
|------|------|------|-------------|
| **유럽은행감독청** | **EBA** | 건전성 강화, 단일 규정집 수립/운영, 회원국 은행부문 위험/취약성 평가 | EU 전역 AML/CFT 당국과의 협력 촉진, RBA 적용방법 공동 이해 촉진 |
| **유럽증권시장감독청** | **ESMA** | 투자자/시장/재무 안정성 위험 평가, 단일규칙 제정, 신용평가기관/거래정보저장소 직접 감독 | AML/CFT 감독 수행 |
| **유럽보험연금감독청** | **EIOPA** | 금융시스템 안전성, 금융시장/상품 투명성, 보험계약자/연금회원/수혜자 보호 | AML/CFT 감독 수행 |
| **자금세탁방지청** | **AMLA** | 2024년 6월 설립, EU AML/CFT 중앙감독기구 | 고위험 금융기관 직접 감독, 국가 감독기관 협력 |

- EBA, ESMA, EIOPA는 AML/CFT를 위해 **공동위원회를 구성**하여 RBA 적용방법 개발 등 공동의 이해 촉진
- 지침 2015/849 및 규정 2015/847의 위임사항 전달 및 표준 수립에 참여

### 5.3 영국(UK)의 자금세탁방지 감독기관

영국은 AML/CFT를 위해 다음 기관을 분리 운영한다:

| 기관 | 약어 | 역할 |
|------|------|------|
| **금융감독청** | **FCA** | 소비자 보호, 금융회사 영업행위 감독. AML/CFT 주도 |
| **건전성감독청** | **PRA** | 은행/보험회사 등 금융회사의 건전성 감독 |
| **자금세탁방지전문감독실** | **OPBAS** | FCA 산하. AML 전문감독자에 대한 AML 감독 일관성 개선 |
| **국가범죄수사청** | **NCA** | UKFIU 및 법집행기관 기능 포함. AML/CFT 업무 수행 |
| **합동자금세탁운영그룹** | **JMLSG** | 영국무역협회 내 구성. 금융서비스업 관련 AML/CFT 우수사례 공표 및 규정 마련 지원 |

**NCA의 주요 AML/CFT 업무**:
- 자금세탁 혐의자에 대한 기소 및 유죄판결 확보
- 범죄자산에 대한 자산 회수 및 압수
- 재무조사관 교육 및 자금세탁 위험 사전 파악
- 영국 금융시스템 남용방지
- 국제적 AML을 위해 다른 국가의 수사기관/FIU와 협력
- **SAR 시스템**을 통한 의심 활동 신고 접수/관리
- UKFIU를 통한 금융정보 분석 및 SARs 데이터 기반 범죄 패턴 파악/수사 지원
