# 의심스러운 거래보고서 (STR) 양식 명세

> **공통사항**: 서식의 입력항목 중 `*`로 표시된 항목은 **선택사항**이다. `*`가 없는 항목은 **필수**이다.
> 단, CDD(고객신원확인의무)를 이행한 고객으로부터 취득한 신원확인정보는 필수 기재하여야 한다.

---

## 표제부

보고서의 일반사항(메타데이터)을 기재한다.

| 번호 | 필드명 | 필수/선택 | 타입 | 설명 |
|------|--------|-----------|------|------|
| 1 | 문서번호 | 필수 | string | 보고기관별 연도 일련번호 (예: `2008-001`) |
| 2 | 보고일자 | 필수 | date | 보고책임자가 금융정보분석원에 실제 보고한 일자 |
| 3 | 정정보고시 종전문서번호 | 선택 | string | 종전 보고서를 정정하는 경우 종전 문서번호 |

## I. 보고기관에 관한 정보

| 번호 | 필드명 | 필수/선택 | 타입 | 설명 |
|------|--------|-----------|------|------|
| 1 | 보고기관명 | 필수 | string | 법률상 명칭 기재 (예: 기업은행 → `중소기업은행`). 전산서식은 기관고유코드 |
| 2 | 보고책임자명 | 필수 | string | 보고책임자 성명 (예: 준법감시인). 외국인은 Last Name, First Name 순 |
| 3 | 보고담당자명 | 필수 | string | 본점 보고담당자 성명 (예: 준법감시실 담당자) |
| 4 | 보고담당자 전화번호 | 필수 | string | 본점 보고담당자 전화번호 |

## II. 의심스러운 거래자에 관한 정보

### II-1. 공통사항

모든 거래자(개인/법인 공통)에 대해 기재한다.

| 번호 | 필드명 | 필수/선택 | 타입 | 설명 |
|------|--------|-----------|------|------|
| 5 | 거래자(사업자)명 | 필수 | string | 개인=성명, 개인사업자=개인성명(사업자명), 법인=법인명, 단체=단체명. 외국인은 Last Name, First Name 순 |
| 6 | 거래자(사업자) 실명번호구분 | 필수 | code | 아래 실명번호구분 코드표 참조 |

| 8 | 거래자(사업자) 국적 | 필수 | code | `1`=한국, `2`=외국 (국명: ISO 2자리 코드, 예: `JP`) |

## 보고 기준 요약

### 보고기준금액

- **외국통화**: 1만 미합중국 달러 또는 그에 상당하는 타 통화 금액
- **원화**: 2,000만원 이상
- 기준금액 이하라도 불법재산 의심 또는 자금세탁/공중협박자금조달 의심 시 보고 가능

### 보고 시기

보고대상 금융거래로 판단하는 때부터 시간적으로 지체 없이 보고한다.

### 보고 방법

온라인, 문서, 전자기록매체(플로피디스크 등). 전자적 방식은 "위험기반 혐의거래보고(STR) 지침"에 따른다.
