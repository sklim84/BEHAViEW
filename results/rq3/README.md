# RQ3 results — Label efficiency (Figure 6, Table 8)

| File | Source script | Content |
|---|---|---|
| `figures/` | (rq-specific) | Mirror of paper figures generated by `scripts/<rq>/gen_*.py` |
| `behaview_{hofinet,amlworld,amlnet}.csv` | `scripts/rq3/run_behaview.sh` (+ HOFINET part of `scripts/run_additional_exp.sh`) | BehaView at train fractions {1%, 5%, 10%} |
| `supervised_{hofinet,amlworld,amlnet}.csv` | `scripts/rq3/run_supervised.sh`, `run_bare_mlp.sh` | General supervised baselines (GCN, GAT, GraphSAGE, MLP, XGBoost, LightGBM, CARE-GNN) at the same fractions |
| `bwgnn_gaga_{hofinet,amlworld,amlnet}.csv` | `scripts/rq3/run_bwgnn_gaga.sh` | BWGNN, GAGA fraud-specific GNNs |
| `caregnn_pcgnn_{hofinet,amlworld,amlnet}.csv` | `scripts/rq3/run_caregnn_pcgnn.sh` | CARE-GNN, PC-GNN |
| `consisgad_{hofinet,amlworld,amlnet}.csv` | `scripts/rq3/run_consisgad.sh` | ConsisGAD |
| `boosting_behav.csv` | `scripts/rq3/run_boosting_behav.sh` | XGBoost / LightGBM with behavioral-only features (HOFINET, "fair" comparison) |

Obsolete predecessors (single-ratio supervised sweep, `label_efficiency.csv` subset) live in `results/_backup/rq3_raw/`.

Figure 6 is generated by `visualize/gen_rq3_labeff_figure.py`, which
reads `behaview_${ds}.csv` and `supervised_${ds}.csv`.
